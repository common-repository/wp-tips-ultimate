Wordpress Tips Ultimate is a tips plugin for your wordpress site like google tips. 


== Description ==

Wordpress Tips Ultimate is a cool tips plugin for wordpress. This plugin will enable google tips like UI in your wordpress site. 

You can add tips & embed them in separate page post or widgets. 

<strong>Other Features</strong>
<ul>
	<li>Super easy Installation. </li>
	<li>Unlimited tips items. </li>
	<li>All Major browser supported. </li>
	<li>Mobile, tablet touch supported.</li>
	<li>Minimalist & lightweight.</li>
</ul>

See live demo here: http://lazypersons.com/plugins/wordpress-tips-ultimate/

== Installation ==

Installing this plugin as regular wordpress plugin.

After install, this plugin enable a custom post called <strong>Tips</strong>. Add category & tips on that custom post. 
Adter adding tips, you are ready to use tips in your post, page, widget or anywhere you like. 

Go to page > Add New. On the editor, use this shortcode <code>[tips category="YOUR CATEGORY NAME"]</code>

Use category title, not category id or slag. Make sure spelling is ok. It is case sensitive. 

If you wanna customize column, use shortcode. Example: <code>[tips category="YOUR CATEGORY NAME" column="2"]</code>. Then it will two column tips.
